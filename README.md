# Σύστημα Διαχείρισης Ξενοδοχείου (Hotel Management System) 🏨
**Εργασία στο μάθημα: "Τεχνολογία Βάσεων Δεδομένων"**

Μια επιτραπέζια εφαρμογή (desktop application) για τη διαχείριση λειτουργιών ξενοδοχείου, η οποία αναπτύχθηκε στο πλαίσιο του μαθήματος "Τεχνολογία Βάσεων Δεδομένων". Το σύστημα επιτρέπει τη διαχείριση κρατήσεων, πελατών και δωματίων μέσω ενός γραφικού περιβάλλοντος Java Swing που συνδέεται με βάση δεδομένων Oracle.

---

## 🚀 Γρήγορη Εκκίνηση (Για τον Αξιολογητή)
Αν θέλετε να τρέξετε την εφαρμογή απευθείας, δεν απαιτείται η χρήση IDE ή η ρύθμιση του κώδικα.

1. Μεταβείτε στην ενότητα **[Releases](../../releases)** του αποθετηρίου.
2. Κατεβάστε το αρχείο `HotelManagement.exe`.
3. Βεβαιωθείτε ότι έχετε εγκατεστημένη τη **Java JRE 8** (ή νεότερη) στον υπολογιστή σας.
4. Εκτελέστε το αρχείο `.exe`. Οι ρυθμίσεις σύνδεσης με τη βάση δεδομένων είναι ενσωματωμένες στο εκτελέσιμο για άμεση λειτουργία.

---

## 🛠️ Λειτουργίες
* **Αυθεντικοποίηση Χρηστών**: Ασφαλές σύστημα εισόδου (Login) για τους υπαλλήλους υποδοχής.
* **Διαχείριση Δωματίων**: Προσθήκη, ενημέρωση και αναζήτηση δωματίων βάσει τύπου και διαθεσιμότητας.
* **Διαχείριση Πελατών**: Εγγραφή νέων πελατών και πλήρης έλεγχος Check-in/Check-out.
* **Αυτοματοποιημένοι Υπολογισμοί**: Υπολογισμός κόστους διαμονής και παρεχόμενων υπηρεσιών σε πραγματικό χρόνο.
* **Προηγμένη Λογική Βάσεων Δεδομένων**: Χρήση PL/SQL Stored Procedures και Triggers για τη διασφάλιση της ακεραιότητας των δεδομένων.

---

## 🏗️ Τεχνολογίες & Εργαλεία
* **Γλώσσα Προγραμματισμού**: Java (JDK 8+)
* **Περιβάλλον GUI**: Java Swing / AWT
* **Βάση Δεδομένων**: Oracle Database (σύνδεση μέσω JDBC)
* **Βιβλιοθήκες (Dependencies)**: 
    * `ojdbc8.jar` (Oracle Driver)
    * `rs2xml.jar` (Μετατροπή ResultSet σε Table Model)
* **Εργαλείο Build**: Apache Ant



---

## 📂 Δομή Φακέλων
* `src/`: Περιέχει όλο τον πηγαίο κώδικα της εφαρμογής (`.java`).
* `build.xml`: Script του Ant για τη μεταγλώττιση και τη δημιουργία του πακέτου.
* `Database_Scripts/`: SQL αρχεία για τη δημιουργία των πινάκων και των διαδικασιών στη βάση.

---

## 🔒 Σημείωση Ασφαλείας
Το αρχείο `config.properties` που περιέχει τα διαπιστευτήρια της βάσης δεδομένων έχει **εξαιρεθεί** από τον δημόσιο κώδικα μέσω του αρχείου `.gitignore`. Για τη διευκόλυνση της αξιολόγησης, οι ρυθμίσεις έχουν ενσωματωθεί στο μεταγλωττισμένο αρχείο `.exe` που βρίσκεται στα Releases.
